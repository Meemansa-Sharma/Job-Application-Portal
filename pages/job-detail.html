About the role

We're looking for a skilled Frontend Developer to join our product team at Infosys. You'll build responsive, accessible interfaces using React and Tailwind CSS, collaborate closely with designers and backend engineers, and integrate with REST APIs to deliver high-quality user experiences.

Responsibilities

  • Build and maintain React components and pages
  • Collaborate with UI/UX designers to implement designs accurately
  • Integrate frontend with backend REST APIs
  • Write clean, maintainable, and well-documented code
  • Participate in code reviews and sprint planning

Requirements

  • Strong knowledge of HTML, CSS, and JavaScript
  • Hands-on experience with React.js
  • Familiarity with Tailwind CSS or similar frameworks
  • Understanding of REST APIs and Git version control
  • Good communication and teamwork skills
